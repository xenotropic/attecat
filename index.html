Attecat

Attendance tracking software in php, sqlite, jquery-ui. Tracks attendance based on a numerical score, as well as daily notes for each student and an overall notepad for each student.

Download this project as a .zip file Download this project as a tar.gz file

This is php/sqlite3 software to keep track of student attendance, which I made with the help of my wife, a middle-school teacher. It is functional, but could use some polishing and more features. Message for a live demo URL.
